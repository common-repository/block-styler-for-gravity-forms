== Description ==

**Style gravity forms with the gutenberg block editor**. Get **real-time feedback** on the style updates within the **super fast gutenberg editor**. 

This plugin provides a **gutenberg block** called **Gravity Form Styler** Refer to screenshot 1. You can embed a gravity form and set all the gravity form shortcode parameters. You can also style the **gravity forms button**'s background, text typography, margin, padding, border and border radius.

We also offer a [premium version](https://wptools.app/wordpress-plugin/gutenberg-block-styler-for-gravity-forms/?utm_source=wp&utm_medium=plugin-page&utm_campaign=guten-gf&utm_content=long-desc) of this plugin. 

Premium version provides additional styling options. Refer screenshot 3 and the overview video below.
